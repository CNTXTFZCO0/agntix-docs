## Code examples

<CodeGroup>

```bash curl
curl https://api.agntix.ai/v1/voice/providers \
  -H "x-api-key: pk_live_••••"
```

```javascript Node.js
const { data: providers } = await fetch('https://api.agntix.ai/v1/voice/providers', {
  headers: { 'x-api-key': 'pk_live_••••' },
}).then(r => r.json());

console.log('Active providers:', providers.map(p => p.name).join(', '));
```

```python Python
import httpx

result = httpx.get(
    "https://api.agntix.ai/v1/voice/providers",
    headers={"x-api-key": "pk_live_••••"},
).json()

for provider in result["data"]:
    status = "active" if provider["isActive"] else "inactive"
    print(f"{provider['name']} ({provider['slug']}) — {status}")
```
