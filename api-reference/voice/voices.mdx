## Code examples

<CodeGroup>

```bash curl
curl "https://api.agntix.ai/v1/voice/voices?language=en" \
  -H "x-api-key: pk_live_••••"
```

```bash curl (filter by provider)
curl "https://api.agntix.ai/v1/voice/voices?provider=elevenlabs&language=en" \
  -H "x-api-key: pk_live_••••"
```

```javascript Node.js
const { data: voices } = await fetch(
  'https://api.agntix.ai/v1/voice/voices?language=en',
  { headers: { 'x-api-key': 'pk_live_••••' } },
).then(r => r.json());

for (const voice of voices) {
  console.log(`${voice.name} (${voice.provider}) — ${voice.gender}`);
}
```

```python Python
import httpx

result = httpx.get(
    "https://api.agntix.ai/v1/voice/voices",
    headers={"x-api-key": "pk_live_••••"},
    params={"language": "en"},
).json()

for voice in result["data"]:
    print(f"{voice['name']} ({voice['provider']}) - {voice['gender']}")
```
