## Code examples

<CodeGroup>

```bash curl
curl "https://api.agntix.ai/v1/voice/models?type=TTS" \
  -H "x-api-key: pk_live_••••"
```

```javascript Node.js
const { data: models } = await fetch(
  'https://api.agntix.ai/v1/voice/models',
  { headers: { 'x-api-key': 'pk_live_••••' } },
).then(r => r.json());

const ttsModels = models.filter(m => m.type === 'TTS');
console.log('TTS models:', ttsModels.map(m => m.name));
```

```python Python
import httpx

result = httpx.get(
    "https://api.agntix.ai/v1/voice/models",
    headers={"x-api-key": "pk_live_••••"},
    params={"type": "TTS"},
).json()

for model in result["data"]:
    print(f"{model['name']} — {model['provider']}")
```
