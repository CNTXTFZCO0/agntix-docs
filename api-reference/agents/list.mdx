## Code examples

<CodeGroup>

```bash curl
curl https://api.agntix.ai/v1/chat/agents \
  -H "x-api-key: pk_live_••••"
```

```bash curl (with filters)
curl "https://api.agntix.ai/v1/chat/agents?page=1&limit=20&isActive=true&search=support" \
  -H "x-api-key: pk_live_••••"
```

```javascript Node.js
const res = await fetch('https://api.agntix.ai/v1/chat/agents?page=1&limit=20', {
  headers: { 'x-api-key': 'pk_live_••••' },
});
const { data, pagination } = await res.json();
console.log(`${pagination.total} agents found`);
```

```python Python
import httpx

resp = httpx.get(
    "https://api.agntix.ai/v1/chat/agents",
    headers={"x-api-key": "pk_live_••••"},
    params={"page": 1, "limit": 20, "isActive": True},
)
result = resp.json()
agents = result["data"]
```
