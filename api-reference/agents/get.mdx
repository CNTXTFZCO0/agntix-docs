## Code examples

<CodeGroup>

```bash curl
curl https://api.agntix.ai/v2/chat/agents/a1b2c3d4-e5f6-7890-abcd-ef1234567890 \
  -H "x-api-key: pk_live_••••"
```

```javascript Node.js
const agentId = 'a1b2c3d4-e5f6-7890-abcd-ef1234567890';

const res = await fetch(`https://api.agntix.ai/v2/chat/agents/${agentId}`, {
  headers: { 'x-api-key': 'pk_live_••••' },
});
const agent = await res.json();
console.log('Tools attached:', agent.agentTools?.length ?? 0);
```

```python Python
import httpx

agent_id = "a1b2c3d4-e5f6-7890-abcd-ef1234567890"

agent = httpx.get(
    f"https://api.agntix.ai/v2/chat/agents/{agent_id}",
    headers={"x-api-key": "pk_live_••••"},
).json()
print("Model:", agent["model"]["name"])
```
