## Code examples

<CodeGroup>

```bash curl
curl "https://api.agntix.ai/v1/chat/tools?page=1&limit=20" \
  -H "x-api-key: pk_live_••••"
```

```bash curl (filter by type)
curl "https://api.agntix.ai/v1/chat/tools?type=API_TOOL&search=balance" \
  -H "x-api-key: pk_live_••••"
```

```javascript Node.js
const { data: tools, pagination } = await fetch(
  'https://api.agntix.ai/v1/chat/tools?page=1&limit=20',
  { headers: { 'x-api-key': 'pk_live_••••' } },
).then(r => r.json());

console.log(`${pagination.total} tools found`);
```

```python Python
import httpx

result = httpx.get(
    "https://api.agntix.ai/v1/chat/tools",
    headers={"x-api-key": "pk_live_••••"},
    params={"page": 1, "limit": 20, "type": "API_TOOL"},
).json()

for tool in result["data"]:
    print(f"{tool['name']} ({tool['type']})")
```
