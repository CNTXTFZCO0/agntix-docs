```bash curl

  -H "x-api-key: pk_live_••••"
```

```javascript Node.js
const sessionId = 'ses_01j3m8xkqp8v2nw4f7g9hk0r';

const session = await fetch(
  `https://api.agntix.ai/v1/chat/chat/sessions/${sessionId}`,
  { headers: { 'x-api-key': 'pk_live_••••' } },
).then(r => r.json());

console.log('State:', session.state);
```

```python Python
import httpx

session_id = "ses_01j3m8xkqp8v2nw4f7g9hk0r"

session = httpx.get(
    f"https://api.agntix.ai/v1/chat/chat/sessions/{session_id}",
    headers={"x-api-key": "pk_live_••••"},
).json()

print("State:", session["state"])
```
