## Code examples

<CodeGroup>

```bash curl
curl -X POST https://api.agntix.ai/v1/chat/chat/sessions \
  -H "x-api-key: pk_live_••••" \
  -H "Content-Type: application/json" \
  -d '{
    "agentId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
    "platform": "SDK",
    "metadata": {
      "userId": "usr_123",
      "locale": "en-US"
    }
  }'
```

```javascript Node.js
const session = await fetch('https://api.agntix.ai/v1/chat/chat/sessions', {
  method: 'POST',
  headers: {
    'x-api-key': 'pk_live_••••',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    agentId: 'a1b2c3d4-e5f6-7890-abcd-ef1234567890',
    platform: 'SDK',
    metadata: { userId: 'usr_123', locale: 'en-US' },
  }),
}).then(r => r.json());

console.log('Session ID:', session.id);
// Send your first message using session.id
```

```python Python
import httpx

session = httpx.post(
    "https://api.agntix.ai/v1/chat/chat/sessions",
    headers={"x-api-key": "pk_live_••••"},
    json={
        "agentId": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
        "platform": "SDK",
        "metadata": {"userId": "usr_123", "locale": "en-US"},
    },
).json()

session_id = session["id"]
print("Session ID:", session_id)
```
