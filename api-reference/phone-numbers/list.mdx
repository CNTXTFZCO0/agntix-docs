## Code examples

<CodeGroup>

```bash curl
curl "https://api.agntix.ai/v1/chat/phone-numbers?page=1&limit=20" \
  -H "x-api-key: pk_live_••••"
```

```javascript Node.js
const { data: numbers } = await fetch(
  'https://api.agntix.ai/v1/chat/phone-numbers',
  { headers: { 'x-api-key': 'pk_live_••••' } },
).then(r => r.json());

for (const n of numbers) {
  console.log(`${n.number} — agent: ${n.agentId ?? 'unassigned'}`);
}
```

```python Python
import httpx

result = httpx.get(
    "https://api.agntix.ai/v1/chat/phone-numbers",
    headers={"x-api-key": "pk_live_••••"},
).json()

for n in result["data"]:
    print(f"{n['number']} — {n.get('agentId', 'unassigned')}")
```
